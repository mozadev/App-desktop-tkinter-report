## Instalación

```bash

# Activar el entorno virtual
# En Windows-poweshell
venv\Scripts\activate
# Instalar las dependencias
pip install -r requirements.txt
# Generar el ejecutable
pyinstaller --onefile --noconsole menu.py
```
